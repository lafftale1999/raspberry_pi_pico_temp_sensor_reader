\doxysubsubsubsubsection{Detailed Description}
Macros to configure temperature, pressure, and humidity oversampling. 

These macros generate bitfields for use in writing to BM280 control registers. Use the following values for oversampling\+:
\begin{DoxyItemize}
\item BM280\+\_\+\+OSRS\+\_\+\+SKIPPED
\item BM280\+\_\+\+OSRS\+\_\+\+X1
\item BM280\+\_\+\+OSRS\+\_\+\+X2
\item BM280\+\_\+\+OSRS\+\_\+\+X4
\item BM280\+\_\+\+OSRS\+\_\+\+X8
\item BM280\+\_\+\+OSRS\+\_\+\+X16
\end{DoxyItemize}

\begin{DoxyNote}{Note}
All macros return a bitfield; they do not write directly to registers. 
\end{DoxyNote}
