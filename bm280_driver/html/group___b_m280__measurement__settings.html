Macro for setting the standby interval. This will decide for how long the BM280 will sleep between taking measurements. (Only effective in BM280_MODE_NORMAL)

This macro generate a bitfield for use in writing to BM280 configuration register. Use the following values:

  • INTERVAL_0_5MS
  • INTERVAL_62_5MS
  • INTERVAL_125MS
  • INTEVAL_250MS
  • INTERVAL_500MS
  • INTERVAL_1000MS
  • INTERVAL_10MS
  • INTERVAL_20MS
