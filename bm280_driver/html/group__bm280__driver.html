Detailed Description

@{

This is a complete driver for the BM280 sensor using I2C communication. It includes JSON support to export measurement data in a structured format.

Example usage:

#include "bm280.h"
bm280_init(bm280_handle_t *handle, 0x76, INTERVAL_1000MS);
bm280_read_data(*handle);
http_post(bm280_get_json(*handle));
