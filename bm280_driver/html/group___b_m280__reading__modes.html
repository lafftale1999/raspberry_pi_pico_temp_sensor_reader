Macro for setting the mode on the BM280.

This macro generate a bitfield for use in writing to the BM280 Control Measure Setting. Use the following values for setting the mode:

  • BM280_MODE_NORMAL
  • BM280_MODE_SLEEP
  • BM280_MODE_FORCED
Parameters
modeUse the Defined BM280_MODE_XXX to assign the correct values.

Bitfield to be written into register.
