Read temperature, pressure, and humidity from the BM280.

This function reads raw temperature, pressure, and humidity values from the sensor and then converts them using Bosch's official compensation formulas. Then parses these values into JSON format.

Parameters
handleBM280 device handle.
Returns
0 on success, 1 on error (e.g., I2C read or conversion failure).
